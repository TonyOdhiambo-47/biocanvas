# BioCanvas

> The Central Dogma, animated. Watch DNA become a protein, step by step.

Three canvases, on one page, with no login and no hidden AI:

1. **DNA replication**. how a cell copies its own genome, semi-conservatively
2. **Transcription**. how DNA is read out as messenger RNA
3. **Translation**. how mRNA is decoded into a protein at the ribosome

Live: https://biocanvas.vercel.app

## Why this exists

Kenyan Senior School Biology (CBC Grade 10 to 12, as of January 2026) covers all three of these processes. Most students encounter them as long text in a textbook, then as an exam question. A short animated explainer bridges the gap for learners who think in pictures, in a way that is free, works on a phone, and does not need signup.

PhET has old Flash-era versions of these sims. Khan Academy has video. Nothing in that space is genuinely mobile-first, free, open source, and tight enough to ship on a slow connection.

## How it handles the biology

Every step is hand-written. Nothing is generated by a language model. The sequence of events follows Alberts *Molecular Biology of the Cell*, 6th edition, and was reviewed against the NCBI Bookshelf. Some deliberate choices:

**Replication.** The 8-step flow covers topoisomerase, helicase, single-strand binding proteins (SSB / RPA), primase, DNA polymerase with leading and lagging synthesis, Okazaki fragments, RNase H / FEN1 primer removal, DNA ligase, and the two daughter helices. Many undergraduate explainers omit SSBs and topoisomerase; we include them because CBC Biology at senior school level explicitly covers them.

**Transcription.** The order of the 7 steps corrects a common textbook simplification: the 5' cap is added first (to the nascent transcript while Pol II is still moving), not last. Splicing happens co-transcriptionally. Poly-A tail comes at the end, after CPSF / CstF cleavage at AAUAAA.

**Translation.** The initiator Met-tRNA enters the P site directly. The three stop codons UAA, UAG, UGA are recognized by a release factor, not a tRNA. The peptidyl transferase center is correctly labelled as a ribozyme activity of the large subunit. The standard genetic code (NCBI Translation Table 1) is hardcoded verbatim.

## Controls

- **Click a step dot** at the top to jump directly to that step
- **Arrow keys** to step through
- **Space** to play / pause
- **1 / 2 / 3** to switch between replication, transcription, and translation
- **Play** auto-advances through the steps

## Running it locally

```bash
git clone https://github.com/TonyOdhiambo-47/biocanvas.git
cd biocanvas
npm install
npm run dev
```

Open http://localhost:5173.

## Deploying to Vercel

The repo ships with a `vercel.json` that is correct for a Vite static build. Either:

- push to GitHub and import the repo in Vercel, or
- from this directory, run `vercel --prod`

There is no backend. No database. Nothing phones home.

## Stack

- Vite with vanilla TypeScript
- Canvas 2D for all animation (no WebGL, no Three.js, no Framer Motion)
- DM Sans and JetBrains Mono from Google Fonts
- Everything else is hand-written

Total bundle (gzipped): roughly 10 KB of JavaScript and 2 KB of CSS. The page loads from cold in under 300 ms on a cheap phone.

## Contributing

PRs welcome. Two rules:

1. If a biological claim changes (e.g., "Okazaki fragments are about 100 to 200 nucleotides"), cite the primary source (Alberts, Lodish, NCBI Bookshelf, or equivalent) in the PR description.
2. No AI-generated content, no hallucinated diagrams. The point of this tool is that every detail is something a biology teacher could verify.
